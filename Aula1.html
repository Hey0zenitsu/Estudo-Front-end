O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, dinâmica e amplamente utilizada tanto por iniciantes quanto por desenvolvedores experientes. Ela é famosa pela sua sintaxe simples e legibilidade, o que facilita o aprendizado e a produtividade. Python é extremamente versátil, sendo utilizada em diversas áreas como:

  • Desenvolvimento web (Flask, Django)
  • Análise de dados (Pandas, NumPy)
  • Inteligência artificial e aprendizado de máquina (TensorFlow, scikit-learn)
  • Automação de tarefas
  • Criação de jogos e interfaces gráficas

Instalando Python

Antes de começar a programar em Python, você precisa instalá-lo em seu computador. Aqui estão os passos para instalação:

  • Acesse o site oficial do Python: python.org/downloads.
  • Escolha a versão recomendada para o seu sistema operacional (Windows, macOS ou Linux).
  • Execute o instalador. Durante a instalação, certifique-se de marcar a opção "Add Python to PATH".
  • Verifique se a instalação foi bem-sucedida abrindo o terminal ou prompt de comando e digitando o comando:
cmd 
        # Verificando se o python está instalado com cmd:
        python --version
C:\User\Project>

Seu primeiro programa Python

Após a instalação, vamos escrever o seu primeiro programa. Abra o editor de texto ou IDE (como VS Code, PyCharm ou até mesmo o IDLE do Python) e crie um novo arquivo chamado primeiro_programa.py. Dentro dele, escreva o seguinte código::

main.py 
# Este é um exemplo de código em Python
def hello_world() :
    print("Hello, VS Code Style!")

hello_world()
C:\User\Project>

Ao executar esse código, a mensagem "Hello, VS Code Style!" será exibida no terminal. Isso é um exemplo de uma função em Python (print()) que exibe uma saída na tela.
