Conversão de Tipos em JavaScript

Em JavaScript, a conversão de tipos permite transformar um valor de um tipo para outro. Isso é útil para manipular dados e garantir que as operações sejam realizadas corretamente.

Conversão Explícita

  • String(): Converte um valor para uma string.
  • Number(): Converte um valor para um número.
  • Boolean(): Converte um valor para um booleano.

Exemplo de Conversão Explícita:

Conversão Explícita: 
const num = 42;
const str = String(num); // "42"
            
const bool = Boolean(0); // false
const numFromString = Number("3.14"); // 3.14
false
3.14

Conversão Implícita:

A conversão implícita acontece automaticamente durante operações. Às vezes, pode gerar resultados inesperados.

Exemplo de Conversão Implícita:

Conversão Implícita 
const result1 = "5" + 2; // "52" (número convertido para string)
const result2 = "5" - 2; // 3 (string convertida para número)
const result3 = true + 1; // 2 (true convertido para 1)
"52"
3
2

Valores que se Comportam Estranhamente

Alguns valores possuem comportamentos específicos durante a conversão:

  • null para número:: 0
  • undefined para número:: NaN (Not-a-Number)
  • Strings vazias para número:: 0

Exemplo:

Valores que se Comportam Estranhamente 
Number(null); // 0
Number(undefined); // NaN
Number(""); // 0
0
NaN
0

Dicas

  • ●: Sempre que possível prefira a conversão explícita para evitar comportamentos inesperados.
  • ●: Use o operador typeof para verificar o tipo de uma variável antes e depois da conversão.

Exemplo com typeof: javascript Copiar Editar

typeof 
const value = "123";
console.log(typeof value); // "string"
            
const numberValue = Number(value);
console.log(typeof numberValue); // "number"
