Console API no JavaScript

O Console API é uma ferramenta poderosa no JavaScript para inspecionar e debugar o código. Ele oferece métodos úteis para registrar mensagens, medir tempos de execução, verificar condições e muito mais. Abaixo, segue um guia prático com os principais métodos do console.

1. console.log()

O método mais usado para imprimir informações no console.

Exemplo:

Uso Do Console.log() 
console.log("Olá, mundo!");
console.log("O valor de x é:", 42);
Olá, mundo!

2. console.error()

Exibe mensagens de erro no console. Útil para destacar problemas.

Exemplo:

Uso Do Console.error() 
console.error("Algo deu errado!");
Olá, mundo!

3. console.warn()

Mostra avisos no console, com menos destaque que os erros.

Uso Do Console.warn() 
console.warn("Essa funcionalidade será removida em breve!");
Olá, mundo!

4. console.table()

Exibe dados tabulares (como arrays ou objetos) no formato de tabela.

Exemplo:

Uso Do Console.table() 
const usuarios = [
  { id: 1, nome: "Bruno", idade: 25 },
  { id: 2, nome: "Ana", idade: 30 }
];
console.table(usuarios);
Olá, mundo!

5. console.group() e console.groupEnd()

Organiza mensagens em grupos, para melhor visualização no console.

Exemplo:

Uso Do Console.group() e Console.groupEnd() 
console.group("Detalhes do Pedido");
console.log("Produto: Notebook");
console.log("Preço: R$3000");
console.groupEnd();
Olá, mundo!

6. console.time() e console.timeEnd()

Usado para medir o tempo de execução de uma operação.

Exemplo: javascript Copiar Editar

Uso Do Console.time() e Console.timeEnd() 
console.time("Tempo de execução");
// Algum código
for (let i = 0; i < 1000000; i++) {}
console.timeEnd("Tempo de execução");
Olá, mundo!

7. Console.assert()

Verifica uma condição. Se for falsa, exibe uma mensagem de erro.

Exemplo:

Uso Do Console.assert() 
const idade = 15;
console.assert(idade >= 18, "Idade mínima não alcançada!");
Olá, mundo!

8. Console.clear()

Limpa todas as mensagens no console.

Exemplo:

Uso Do Console.clear() 
console.clear();
Olá, mundo!

E assim encerramos a aula sobre a Console API no JavaScript! 🎉

Com essas ferramentas, você pode debugar, medir desempenho e inspecionar seus dados de forma mais eficiente. Pratique bastante e explore cada método para dominar o uso do console!
