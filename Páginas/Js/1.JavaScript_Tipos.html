Tipos de Dados em JavaScript

No JavaScript, temos alguns tipos de dados fundamentais que são muito usados no dia a dia da programação:

Strings (String)

As strings são sequências de caracteres usadas para representar texto. Elas podem ser criadas com aspas simples ou duplas.

  • Exemplo de string com aspas duplas: "Olá Mundo!"
  • Exemplo de string com aspas simples: 'JavaScript'

Você pode manipular strings com várias funções, como:

script.js 
// Manipulação de Strings em JavaScript
let texto = "Olá, Mundo!";
let tamanho = texto.length; // Retorna o tamanho da string
let maiuscula = texto.toUpperCase(); // Converte para maiúsculas

Números (Number)

Os números em JavaScript podem ser inteiros (sem casas decimais) ou de ponto flutuante (com casas decimais).

  • Exemplos de números inteiros: 10 -5 100
  • Exemplos de números de ponto flutuante: 3.14 -0.5 2.5
script.js 
// Operações Matemáticas em JavaScript
let soma = 10 + 5; // 15
let produto = 3 * 4; // 12
let divisao = 10 / 2; // 5

Booleanos (Boolean)

Os booleanos representam dois valores possíveis: true (verdadeiro) ou false (falso). Eles são usados para decisões lógicas e controles de fluxo (como em estruturas if).

  • Exemplo de valor booleano verdadeiro: true
  • Exemplo de valor booleano falso: false
script.js 
// Operações Booleanas em JavaScript
let maiorQueCinco = 10 > 5; // true
let igual = 5 === 3; // false

Esses são os três tipos primitivos que você vai utilizar com frequência na programação em JavaScript. Com as strings, números e booleanos, você já pode criar variáveis e fazer operações simples para construir seus programas.
