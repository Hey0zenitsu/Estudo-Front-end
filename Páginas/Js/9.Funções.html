Funções em JavaScript

O que são funções?

Funções em JavaScript são blocos de código reutilizáveis que podem ser chamados para realizar uma tarefa específica. Elas ajudam a organizar e reduzir a repetição de código, tornando o programa mais eficiente e fácil de manter.

Como declarar uma função?

Sintaxe básica

A declaração de uma função pode ser feita utilizando a palavra-chave function, seguida pelo nome da função, parênteses (que podem conter parâmetros) e chaves para delimitar o bloco de código.

Exemplo 
function nomeDaFuncao(param1, param2) {
    // Código a ser executado
}

Exemplo:

Exemplo 
function saudacao(nome) {
    console.log("Olá, " + nome + "!");
}
saudacao("Bruno");

Tipos de funções

1. Funções Declaradas

São aquelas definidas usando a palavra-chave function e possuem um nome. Elas são elevadas (hoisting), ou seja, podem ser chamadas antes de serem declaradas.

Exemplo 
function somar(a, b) {
    return a + b;
}
console.log(somar(2, 3)); // 5
1

2. Funções Anônimas

São funções sem nome, geralmente atribuídas a uma variável.

Exemplo 
const multiplicar = function(a, b) {
    return a * b;
};
console.log(multiplicar(2, 3)); // 6
1

3. Funções de Setas (Arrow Functions)

São uma forma mais concisa de escrever funções. Elas não possuem seu próprio this.

Exemplo 
const dividir = (a, b) => a / b;
console.log(dividir(6, 3)); // 2
1

Exemplo

    console.log("Bem-vindo, " + nome + "!");
}
saudacao(); // "Bem-vindo, Visitante!"

Escopo

Escopo Local

Uma variável declarada dentro de uma função só pode ser acessada dentro dela.

Exemplo 
function exemplo() {
    let mensagem = "Olá!";
    console.log(mensagem); // Funciona
}
console.log(mensagem); // Erro
1

Escopo Global

Variáveis declaradas fora de qualquer função podem ser acessadas em qualquer parte do código.

Exemplo 
let mensagem = "Global!";
function exemplo() {
    console.log(mensagem); // "Global!"
}
exemplo();
1

Funções como Valores

Em JavaScript, funções podem ser atribuídas a variáveis, armazenadas em arrays ou passadas como argumentos para outras funções.

Exemplo 
const dobro = (x) => x * 2;

function aplicarFuncao(numero, funcao) {
    return funcao(numero);
}

console.log(aplicarFuncao(5, dobro)); // 10
1

Funções Aninhadas

Uma função pode estar contida dentro de outra função, permitindo a criação de escopos mais restritos.

Exemplo 
function externa() {
    function interna() {
        console.log("Sou uma função interna!");
    }
    interna();
}
externa(); // "Sou uma função interna!"
1

Conclusão

Funções são uma das ferramentas mais poderosas do JavaScript. Elas permitem modularizar o código, reduzir redundâncias e criar programas mais organizados e reutilizáveis. Pratique bastante para dominar os diferentes tipos e usos de funções!
