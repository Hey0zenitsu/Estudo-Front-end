Truty and Falsy (boolean)

O tipo de dado Booleano é usado para representar valores lógicos, geralmente expressando duas condições mutuamente exclusivas: verdadeiro ou falso. As duas formas mais comuns de representação são true (verdadeiro) e false (falso), mas o conceito por trás do tipo Booleano pode ser ampliado de outras formas, dependendo do contexto ou da linguagem de programação. Aqui estão algumas formas alternativas de se determinar o verdadeiro e o falso:

  • 1. Representações Padrão: true e false
  • 2. Valores Numéricos: 0 e 1
  • 3. Strings: "" Representa false e com texto Representa true
  • 4. Objetos e Arrays: {} Representa true [] Representa true
  • 5. Valores Especiais: null Representa false undefined Representa false
  • 6. Função Boolean(): Boolean(0) false Boolean(texto) true
  • 7. Operadores Lógicos e Comparações: && Retorna true se ambos os operandos forem true || Retorna true se pelo menos um dos operandos for true
  • 8. Valores truthy e falsy.: 0 falsy e texto truthy.

Representações Padrão:

Representações Padrão 
let valorVerdadeiro = true;
let valorFalso = false;

console.log(valorVerdadeiro); // Exibe: true
console.log(valorFalso);      // Exibe: false
true
false

Essas variáveis representam os dois valores possíveis de um tipo de dado booleano em JavaScript, que são usados para controlar a lógica do programa.

Valores Numéricos:

Representações Padrão 
let valorZero = 0;
let valorPositivo = 1;
let valorNegativo = -1;

console.log(valorZero);        // Exibe: 0 (falsy)
console.log(valorPositivo);    // Exibe: 1 (truthy)
console.log(valorNegativo);    // Exibe: -1 (truthy)
falsy
truthy
truthy

O código define três variáveis numéricas: valorZero com o valor 0, valorPositivo com o valor 1 e valorNegativo com o valor -1. Em seguida, imprime esses valores no console, mostrando que 0 é tratado como falsy e 1 e -1 como truthy.

Strings:

Strings 
let stringVazia = "";
let stringTexto = "texto";

console.log(stringVazia);   // Exibe: "" (falsy)
console.log(stringTexto);   // Exibe: "texto" (truthy)
falsy
truthy

O código define duas variáveis de string: stringVazia, que é uma string vazia "", e stringTexto, que contém o valor "texto". Ao exibir os valores no console, a string vazia é tratada como falsy (falsa), enquanto a string "texto" é tratada como truthy (verdadeira).

Objetos e Arrays:

Objetos e Arrays: 
let objetoVazio = {};
let arrayVazio = [];
let objetoComDados = { nome: "João" };
let arrayComDados = [1, 2, 3];

console.log(objetoVazio);       // Exibe: {} (truthy)
console.log(arrayVazio);        // Exibe: [] (truthy)
console.log(objetoComDados);    // Exibe: { nome: "João" } (truthy)
console.log(arrayComDados);     // Exibe: [1, 2, 3] (truthy)
falsy
truthy

O código define quatro variáveis: objetoVazio (um objeto vazio), arrayVazio (um array vazio), objetoComDados (um objeto com um dado) e arrayComDados (um array com três números). Ao exibir os valores no console, tanto objetos vazios quanto objetos ou arrays com dados são tratados como truthy (verdadeiros), ou seja, são considerados "verdadeiros" em contextos lógicos.

Valores Especiais:

Valores Especiais 
let valorNull = null;
let valorUndefined = undefined;
let valorNaN = NaN;

console.log(valorNull);       // Exibe: null (falsy)
console.log(valorUndefined);  // Exibe: undefined (falsy)
console.log(valorNaN);        // Exibe: NaN (falsy)
falsy
falsy
falsy

O código define três variáveis com valores especiais: valorNull (null), valorUndefined (undefined) e valorNaN (NaN). Ao exibir esses valores no console, todos são tratados como falsy (falsos), ou seja, são considerados "falsos" em contextos lógicos.

Função Boolean()

Função Boolean() 
let valorZero = 0;
let valorTexto = "texto";

console.log(Boolean(valorZero));    // Exibe: false
console.log(Boolean(valorTexto));   // Exibe: true
falsy
truthy

O código utiliza a função Boolean() para converter valores em seu equivalente booleano. A função retorna false para valores falsy e true para valores truthy.

  • Boolean(valorZero): converte 0 para false pois 0 é falsy.
  • Boolean(valorTexto): converte "texto" para true pois strings não vazias são truthy.

Operadores Lógicos e Comparações:

Operadores Lógicos e Comparações: 
let a = true;
let b = false;

console.log(a && b);  // Exibe: false (ambos precisam ser true para retornar true)
console.log(a || b);  // Exibe: true (pelo menos um precisa ser true para retornar true)
console.log(!a);      // Exibe: false (inverte o valor de true para false)
falsy
truthy

O código utiliza operadores lógicos para realizar comparações entre as variáveis a e b, que possuem os valores true e false, respectivamente.

  • a && b: O operador && (E lógico) retorna true somente se ambos os operandos forem true. Neste caso como b é false a saída é false
  • a || b: O operador || (OU lógico) retorna true se pelo menos um dos operandos for true. Como a é true a saída é true.
  • !a: O operador ! (não lógico) inverte o valor de a que é true transformando-o em false.

Valores truthy e falsy:

Operadores Lógicos e Comparações: 
let valorFalsy = 0;
let valorTruthy = "texto";

console.log(Boolean(valorFalsy));  // Exibe: false (0 é falsy)
console.log(Boolean(valorTruthy)); // Exibe: true ("texto" é truthy)
falsy
truthy

O código utiliza a função Boolean() para avaliar os valores de valorFalsy e valorTruthy, que representam um valor falsy e um valor truthy, respectivamente.

Na aula, aprendemos sobre valores truthy e falsy em JavaScript, que determinam se algo é considerado "verdadeiro" ou "falso" em contextos lógicos.

Vimos também como a função Boolean() converte valores para seu equivalente booleano e como os operadores lógicos (&&, ||, !) funcionam em expressões. Esses conceitos são essenciais para controle de fluxo em programação.
