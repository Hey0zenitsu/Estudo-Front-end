Funções em JavaScript

O que são funções?

Funções em JavaScript são blocos de código reutilizáveis que podem ser chamados para realizar uma tarefa específica. Elas ajudam a organizar e reduzir a repetição de código, tornando o programa mais eficiente e fácil de manter.

Como declarar uma função?

Sintaxe básica

A declaração de uma função pode ser feita utilizando a palavra-chave function, seguida pelo nome da função, parênteses (que podem conter parâmetros) e chaves para delimitar o bloco de código.

Exemplo: Declaração de Função 
function nomeDaFuncao(param1, param2) {
    // Código a ser executado
            }
1

Este exemplo mostra a declaração básica de uma função em JavaScript. Ela pode receber parâmetros, como `param1` e `param2`, e executar um bloco de código quando chamada.

Exemplo:

Exemplo: Função de Saudação 
function saudacao(nome) {
    console.log("Olá, " + nome + "!");
            }
saudacao("Bruno");
1

Este exemplo define uma função chamada `saudacao` que recebe um parâmetro `nome` e exibe uma mensagem de saudação no console. Quando a função é chamada com o argumento "Bruno", ela imprime "Olá, Bruno!" no terminal.

Tipos de funções

1. Funções Declaradas

São aquelas definidas usando a palavra-chave function e possuem um nome. Elas são elevadas (hoisting), ou seja, podem ser chamadas antes de serem declaradas.

Exemplo: Função de Soma 
function somar(a, b) {
    return a + b;
            }
console.log(somar(2, 3)); // 5
1

Este exemplo define uma função chamada `somar`, que recebe dois parâmetros `a` e `b`, e retorna a soma desses valores. Quando a função é chamada com os argumentos `2` e `3`, o resultado `5` é impresso no console.

2. Funções Anônimas

São funções sem nome, geralmente atribuídas a uma variável.

Exemplo: Função de Multiplicação 
const multiplicar = function(a, b) {
    return a * b;
            };
console.log(multiplicar(2, 3)); // 6
1

Este exemplo mostra como declarar uma função anônima chamada `multiplicar`, que recebe dois parâmetros e retorna o produto desses números. Quando a função é chamada com os argumentos `2` e `3`, o resultado `6` é impresso no console.

3. Funções de Setas (Arrow Functions)

São uma forma mais concisa de escrever funções. Elas não possuem seu próprio this.

Exemplo: Função de Divisão 
const dividir = (a, b) => a / b;
console.log(dividir(6, 3)); // 2
1

Esse exemplo mostra como declarar uma função de divisão utilizando a sintaxe de arrow function. Quando a função `dividir(6, 3)` é chamada, o resultado `2` é exibido no console.

Parâmetros e Argumentos

Parâmetros

Exemplo: Função de Saudação 
function cumprimentar(nome) {
    console.log("Olá, " + nome + "!");
}
cumprimentar("Bruno")
1

Esse código JavaScript define uma função chamada cumprimentar(), que recebe um nome e imprime uma saudação personalizada no terminal. Ao chamar cumprimentar("Bruno"), a mensagem "Olá, Bruno!" será exibida no terminal.

Argumentos

Os valores reais passados para os parâmetros quando a função é chamada. javascript Copiar Editar

Exemplo: Chamando a Função 
cumprimentar("Bruno");  // "Olá, Bruno!"
1

Esse exemplo demonstra como chamar uma função em JavaScript. Ao executar o código, o resultado será uma saudação personalizada: "Olá, Bruno!"

Parâmetros Padrão

Se nenhum valor for passado, os parâmetros podem assumir um valor padrão. javascript Copiar Editar

Exemplo: Função com Valor Padrão 
function saudacao(nome = "Visitante") {
    console.log("Bem-vindo, " + nome + "!");
            }
saudacao(); // "Bem-vindo, Visitante!"
1

Este exemplo demonstra uma função com um valor padrão para o parâmetro `nome`. Quando não é passado um argumento, a função utiliza o valor `"Visitante"`.

Escopo

Escopo Local

Uma variável declarada dentro de uma função só pode ser acessada dentro dela.

Exemplo: Escopo de Variáveis 
function exemplo() {
    let mensagem = "Olá!";
    console.log(mensagem); // Funciona
            }
console.log(mensagem); // Erro
1

Este exemplo mostra a diferença entre o escopo de uma variável dentro de uma função e fora dela. A variável `mensagem` é definida dentro da função `exemplo`, então ela não é acessível fora dessa função, resultando em erro.

Escopo Global

Variáveis declaradas fora de qualquer função podem ser acessadas em qualquer parte do código.

Exemplo: Variável Global 
let mensagem = "Global!";
function exemplo() {
    console.log(mensagem); // "Global!"
            }
exemplo();
1

Este exemplo mostra como uma variável global, como `mensagem`, pode ser acessada dentro de uma função. Mesmo dentro de `exemplo`, a variável `mensagem` mantém seu valor e pode ser usada sem problemas.

Funções como Valores

Em JavaScript, funções podem ser atribuídas a variáveis, armazenadas em arrays ou passadas como argumentos para outras funções.

Exemplo 
const dobro = (x) => x * 2;
            
function aplicarFuncao(numero, funcao) {
    return funcao(numero);
            }
            
console.log(aplicarFuncao(5, dobro)); // 10
1

Funções Aninhadas

Uma função pode estar contida dentro de outra função, permitindo a criação de escopos mais restritos.

Exemplo 
function externa() {
    function interna() {
        console.log("Sou uma função interna!");
                }
    interna();
            }
externa(); // "Sou uma função interna!"
1

Conclusão

Funções são uma das ferramentas mais poderosas do JavaScript. Elas permitem modularizar o código, reduzir redundâncias e criar programas mais organizados e reutilizáveis. Pratique bastante para dominar os diferentes tipos e usos de funções!
