Acessando e Alterando Elementos em Arrays

Acessando Elementos

Os elementos de um array são acessados por meio do índice (começando do 0).

Card.Exemplo 
let frutas = ['maçã', 'banana', 'laranja'];

console.log(frutas[0]); //  saída: maçã
console.log(frutas[1]); //  saída: banana
C:UserProject>

'maçã'
'banana'

Alterando Elementos

Você pode alterar o valor de um elemento do array diretamente, acessando pelo índice. javascript Copiar
Card.Exemplo 
let frutas = ['maçã', 'banana', 'laranja'];

frutas[1] = 'uva';
console.log(frutas); //  saída: ['maçã', 'uva', 'laranja']
C:UserProject>

['maçã', 'uva', 'laranja']
